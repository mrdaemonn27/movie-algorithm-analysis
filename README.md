# Movie Algorithm Analysis

This project is an interactive **movie management and algorithm analysis system**
built using **Python and Streamlit**.  
It implements **sorting, searching, and performance benchmarking algorithms**
on movie datasets.

---

## Features
- Movie data management from CSV files
- Sorting algorithms:
  - Quick Sort (Iterative & Recursive)
  - Insertion Sort (Iterative & Recursive)
- Searching algorithm:
  - Binary Search
- Algorithm performance benchmarking
- Interactive web-based interface using Streamlit

---
